#### 文件

***GET*** /v3/file/{identity}
***POST*** /v3/file/get
```
GET /v3/file/1820b058795b4297b338ba066eea9319
```

或者

```
POST /v3/file/get
```
```json
{
    "identity": "1820b058795b4297b338ba066eea9319"
}
```
```json
{
    "path": "/lol"
}
```
