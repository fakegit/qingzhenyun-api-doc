# 清空回收站

> 将回收站中的文件清空

> 如果复制的为文件夹而不是文件，操作为异步操作，执行操作后目标文件夹将会被锁定

#### 复制文件或文件夹使用的参数

|  参数  	|  类型  	| 可选 	|           说明           	|
|:------:	|:------:	|:----:	|:------------------------:	|
|  ts  	| int 	|  是  	|  时间戳 	|



```POST``` ```/v3/newtrash/clear``` ```/v3/newtrash/truncate```


```json
{
	"t":1234
}
```




#### 注意

* 如果文件夹内文件较多，则时间相对较长

#### 响应

> 文件复制成功

```json
{
    "identity": "clear_trash_bsefjaekepp0qvba61k0_5",
    "data": 232,
    "async": false
}
```
