# 列出回收站条目
件

> 列出当前回收站内容

```POST /v3/newtrash/list 或者 GET /v3/newtrash

#### 列出回收站

|  参数  	|  类型  	| 可选 	|           说明           	|
|:------:	|:------:	|:----:	|:------------------------:	|
|  skip  	| int64	|  是  	|        跳过记录数，从0开始    	|
| limit 	| int64 	|  是  	| 返回最大记录数，默认20 	|


```POST``` ```/v3/newtrash/list```


```json
{
	"skip": 199,
	"limit": 50,
	"orderby": [["ctime", "asc"], ["directory", "desc"]]
}
```

或者

```GET``` ```/v3/newtrash?skip=0&limit=4```
