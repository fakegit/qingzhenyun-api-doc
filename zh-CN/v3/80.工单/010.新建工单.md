# 新建工单


```POST```
```/v3/ticket/create``` 或 ```/v3/ticket```

参数

* title 标题
* message 内容
* type 类型（默认10） 【10: 一般问题，20: 文件故障，30: 支付问题，30: 离线下载 100: 其他问题】
* images 附带图片，使用文件哈希（不建议使用）

```json
{
	"title": "apply for professor",
	"message": "微电子工业的发展"
}
```

接受

```json
{
    "identity": 1,
    "userIdentity": 5,
    "title": "apply for professor",
    "replyUserIdentity": 0,
    "type": 10,
    "status": 100,
    "createTime": 1591517119478,
    "refreshTime": 0,
    "message": "微电子工业的发展",
    "images": ""
}
```
